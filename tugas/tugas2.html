Tugas 1: Konsep Sistem Operasi

Pengertian Sistem Operasi

Sistem operasi adalah perangkat lunak yang mengelola perangkat keras komputer dan menyediakan layanan umum untuk program komputer. Sistem operasi menjadi jembatan antara pengguna dan perangkat keras komputer.

Fungsi Sistem Operasi

Beberapa fungsi utama dari sistem operasi meliputi:

  • Manajemen proses
  • Manajemen memori utama
  • Manajemen perangkat penyimpanan sekunder
  • Manajemen input/output
  • Manajemen file
  • Manajemen sistem keamanan dan proteksi

Komponen Utama Sistem Operasi

Komponen utama sistem operasi biasanya mencakup:

  • Kernel: Inti dari sistem operasi yang mengontrol seluruh operasi dasar komputer.
  • Shell: Antarmuka yang memungkinkan pengguna berinteraksi dengan sistem operasi.
  • File System: Struktur yang mengelola penyimpanan data di dalam komputer.
